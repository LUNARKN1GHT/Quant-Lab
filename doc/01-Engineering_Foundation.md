# 项目骨架搭建

## `pyproject.toml` 的编写

之前都是用 `conda` 来管理项目的包。这是第一次使用 `uv` 来管理包。算是全新的体验。只需要在命令行里面写：

```cmd
uv init --python 3.XX

uv add ~ ~ ~

uv remove ~ ~ ~
```

相应的包就会自动输入到我们的项目文档里面，而且已经指定好了版本。相比之前的 `conda`，这个在迁移的时候不知道方便了多少。

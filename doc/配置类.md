# 配置类

## 使用嵌套 dataclass

我们对不同的模块需要不同的配置：

- `Config`:
  - `DataConfig`：程序使用的数据配置
  - `FactorConfig`：因子计算的配置
  - `BacktestConfig`：回测框架使用的配置

如果把下面的子配置平铺下，我们每次传入配置的时候，都会传入其他模块的相关配置，这对相应模块的计算是没必要的。甚至有可能导致“串线”。

## `field` 使用

`field` 是 `dataclass` 开辟时的自动化处理，用 `= [20, 60, 120]` 是硬编码成一个 `list`，但我们实际上只需要一个 `int`.

## `get` 获取配置字典

`get` 里面的空的 `{}` 是一个默认返回值。如果直接用 `["data"]` 获取，可能会因为空值导致无法正常读取，也不方便我们差错。
